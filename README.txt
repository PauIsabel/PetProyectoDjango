INFORMACIÓN DE USO

Esta parte del proyecto esta con Oracle.

Seguir los siguientes pasos.


Esta disponible el scrip.slq de la base de datos para cargar las tablas. llamado Script_BD_PetsDjango >>> cargar desde sql developer

>>>Registre Usuario en pestaña Registrarse

>>>Inicie sesión con las credenciales mencionadas.

>>>Agregar productos> ir a pestaña ADMIN y luego a agregar productos y complete los datos solicitados, cargue una imagen obligatoria para el producto a ingresar y registrelo.

>>>Las imagenes para cargar puede seleccionarlas desde carpeta "Cargar Imagen producto" ( son solamente para usarlas para cargar, las que quedan en el proyecto se almacenan en media

>>>Listar producto> en la pestaña ADMIN liste los productos que estan en la base de datos.

>>>TIENDA> Desde el menu tienda puede realizar compra de productos y agregarlos al carrito.
